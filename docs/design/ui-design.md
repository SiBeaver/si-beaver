# si-beaver Dashboard UI 设计草案

## 定位

纯展示面板（只读），数据操作通过 AI 工具 / REST API 完成。
面向开发者，日常打开快速掌握项目认知状态。

---

## 信息架构

### 页面结构

```
┌─────────────────────────────────────────────┐
│  侧边栏导航（窄）        主内容区              │
│  ┌──────┐  ┌──────────────────────────────┐ │
│  │ 概览 │  │                              │ │
│  │ 路线图│  │    当前选中页面的内容          │ │
│  │ 风险 │  │                              │ │
│  │      │  │                              │ │
│  └──────┘  └──────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 三个页面

| 页面 | 核心问题 | 展示内容 |
|------|---------|---------|
| **概览** | "项目整体怎么样" | 关键数字卡片 + 最近活动 |
| **路线图** | "目标和进度" | 目标树 + 子项进度条 |
| **风险** | "有什么问题" | 阻塞项 + 风险矩阵 + 技术债 |

### 概览页数据卡片

- 活跃目标数 / 已达成
- 待办任务数
- 未解决风险数
- 技术债数
- 最近 5 条活动事件

## 视觉风格

### 关键词

- **Notion 风** — 圆润、宽松、友好、克制
- **自适应主题** — 浅色/深色跟随系统
- **信息密度适中** — 不挤不空，留白有呼吸感
- **色彩克制** — 基础灰色为主，颜色仅用于状态/强调

### 配色思路

| 用途 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 背景 | #ffffff / #f8f9fa | #1a1a1a / #252525 |
| 卡片 | #ffffff 带微阴影 | #2a2a2a |
| 文字主色 | #1a1a1a | #e5e5e5 |
| 文字次色 | #6b7280 | #9ca3af |
| 强调色 | #2563eb (蓝) | #3b82f6 |
| 成功 | #16a34a | #4ade80 |
| 警告 | #d97706 | #fbbf24 |
| 危险 | #dc2626 | #f87171 |

### 圆角 & 间距

- 卡片圆角: 12px
- 按钮/标签圆角: 8px
- 页面内边距: 32px
- 卡片间距: 16px
- 行间距宽松: 1.6

### 字体

系统字体栈，无自定义字体加载：
```
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;
```
